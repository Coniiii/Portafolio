Proyectos Académicos

Recorrido del Caballo en Ajedrez

Implementación del recorrido del caballo en un tablero de ajedrez mediante técnicas de búsqueda y optimización algorítmica.

Python · Algoritmos · Backtracking/BFS

Aplicación de votaciones

Aplicación móvil para Android que usa autenticación segura con Google mediante Firebase y consume una API externa para gestionar el flujo de votación. Incluye manejo de estados, persistencia de sesión y una arquitectura por capas.

Flutter · Firebase (Authentication) · Arquitectura Limpia

Predicción de cultivos en la Araucanía

Implementación de modelos avanzados de Machine Learning para predecir el rendimiento de trigo, con EDA, ajuste de hiperparámetros y análisis de interpretabilidad para apoyar la agricultura de precisión.

Python · Redes Neuronales (MLP) · Modelos Ensemble

Taller de Análisis de Algoritmos Backtracking

Resolución del problema de colocar piezas de dominó en un tablero garantizando cortes en todas las filas y columnas. Se implementó Backtracking para explorar configuraciones válidas, verificando restricciones y exportando la solución final a Excel.

Python · Backtracking · Algoritmos

Python · Grafos · Simulación
